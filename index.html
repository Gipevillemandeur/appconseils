📘 Guide d'utilisation

⚠️ Pas de sauvegarde permanente : Dès que vous fermez l'onglet, les données sont effacées. Ne fermez pas avant d'avoir enregistré votre PDF !

1. Écran d'accueil : Sélectionnez votre classe, le trimestre et la date, puis cliquez sur "Commencer". Les données de votre classe se chargent automatiquement.

2. Saisie (partie JAUNE ✍️) : Cochez les professeurs présents (✓ = présent, ✗ = absent). Remplissez les participants et les observations. Tout se reporte automatiquement dans l'aperçu gris.

3. Listing élèves : Cliquez sur ce bouton pour ouvrir une feuille de suivi élève imprimable.

4. Enregistrement (étape CRUCIALE 🚀) : Cliquez sur "Imprimer compte rendu", puis choisissez "Enregistrer au format PDF". Transmettez ce PDF au GIPE par email.
