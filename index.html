📘 Guide d'utilisation

⚠️Avertissements⚠️ Pas de sauvegarde permanente : Pour votre sécurité, dès que vous fermez l'onglet ou le site, les données sont effacées. Ne fermez pas votre page avant d'avoir enregistré votre PDF ! Mise en page : Le document est configuré pour tenir sur deux pages avec un rappel de l'en-tête sur la seconde.

1. Préparation et SécuritéChoix de la classe : Commencez par sélectionner votre classe dans le menu déroulant. Accès sécurisé : Un code vous sera demandé (Code transmis par le GIPE). Une fois validé, la liste des professeurs et des matières s'affichera automatiquement. Infos générales : Sélectionnez la Date du conseil, le Trimestre et le nom du Président de séance.

2. Saisie des informations (Partie JAUNE ✍️) Équipe pédagogique : Cochez simplement les professeurs présents par defaut ils sont présents, en appuyant sur la coche ils passent en absents. Participants : Saisissez les noms des parents délégués et élèves délégués (un par ligne). Observations : Remplissez les blocs de texte pour la direction, le professeur principal, les élèves et les parents. Note : Tout ce que vous tapez dans la partie JAUNE se reporte automatiquement dans la partie GRISE (Aperçu) à droite. C'est cet aperçu qui sera imprimé.

3. Outils annexes Bouton "Listing élèves" : Si vous avez besoin d'une feuille pour noter pendant le conseil, cliquez sur ce bouton. Vous pouvez l'imprimer vierge ou remplir et cocher directement les cases (Félicitations, Compliments, etc.).

4. Enregistrement et Envoi (Etape CRUCIALE 🚀) Une fois votre saisie terminée : Cliquez sur le bouton "Imprimer compte rendu". Dans la fenêtre qui s'ouvre (Destination), ne choisissez pas votre imprimante papier. Sélectionnez "Enregistrer au format PDF". Enregistrez le fichier sur votre ordinateur (par exemple : CR_3A_T1.pdf). Envoi : Transmettez ce fichier PDF par email au GIPE.
